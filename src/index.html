ValueEdge Beautifier

This extension provides extra UX beautification to the ValueEdge GUI. The beautifications are inspired from Atlassian JIRA.

Features

Coloured "Phase" badges

Phases

  • New
  • Ready
  • Planned
  • In Progress
  • Code Review
  • In Testing
  • Tested
  • Implemented
  • Fixed
  • Done
  • Completed
  • Cancelled
  • Pending Support
  • Awaiting Decision
  • Deferred
  • Proposed Rejected
  • Rejected
  • Duplicate

Views

  • Dashboard view
  • Backlog view
  • Team Backlog view
  • Story/Issue view
